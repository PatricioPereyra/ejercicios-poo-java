# Ejercicios de Programación Orientada a Objetos en Java

Repositorio con ejercicios prácticos de POO en Java, de la materia
Programación II de la Tecnicatura Universitaria en Programación de la UTN-FRA.

## Ejercicios Completados

### Ejercicio D.02 - Tinta y Pluma
Sistema de gestión de tintas con clases sobrecargadas.

**Características:**
- Enumerados `Color` y `Tipo`
- Clase `Tinta` con constructores sobrecargados
- Clase `Pluma` con constructores sobrecargados
- Métodos `add()` y `remove()`
- Uso de StringBuilder
